Задание 1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.

Решение. В колекциях Redis хранятся только уникальные значения. Подсчитать количество посещений с определенных адресов при помощи колекций Redis не возможно.

Задание 2. При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, поиск электронного адреса пользователя по его имени.

В Redis данные хранятся по принципу «ключ-значение». Следовательно нельзя производить поиск и имени и email для одного набора ключ-значение. Если мы внесем строку
set dima@mail.ru dima
то сможем найти только имя по e-mail, так как email выступит ключом. Для обратного поиска необходимо внести еще одну запись вида:
set dima dima@mail.ru
